关键术语小词典

EMIB-T

Intel EMIB 的 TSV bridge 方向，用 embedded bridge 缩短信号/电源路径；重点看 bridge parasitic、PDN 与热机械可靠性。

Known-good optical engine

CPO 中先筛好的光引擎单元。没有 KGOE，后续封装和系统级维修成本会被缺陷放大。

Runtime recalibration

UCIe 3.0 中用于运行期链路调谐的机制，目标是改善功耗和可靠性，但会增加验证状态空间。

CLIPGen

面向 2.5D 架构探索的 chiplet link IP 建模/生成框架，用来在早期比较 package 与互连方案。

关键术语小词典

EMIB-T

Intel EMIB 的 TSV bridge 方向，用 embedded bridge 缩短信号/电源路径；重点看 bridge parasitic、PDN、thermal path 与机械可靠性。

Known-good optical engine

CPO 中先筛好的光引擎单元。没有 KGOE，后续封装、系统级维修和 RMA 成本会被缺陷放大。

Runtime recalibration

UCIe 3.0 中用于运行期链路调谐的机制，目标是改善功耗和可靠性，但会增加验证状态空间。

Detachable fiber attachment

可拆卸光纤连接方案，直接影响 CPO 的测试、返修和现场维护，不能只按初始插损评估。

Package-aware DSE

在架构探索期同步估计 link、routing、PDN、thermal、area 与 test trade-off，而不是等封装定案后补救。

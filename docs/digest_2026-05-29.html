关键术语小词典

  • τ / time constant：这里可理解为系统完成关键数据移动或响应的时间代价。
  • LogicFolding：通过 3D integration 和系统互连优化，把逻辑单元重新组织以降低延迟和数据搬运。
  • Hybrid bonding：更细 pitch、更高密度 die-to-die / wafer-to-wafer 互连的关键工艺。
  • PDN：Power Delivery Network，决定高速芯片和封装的电源完整性。
  • Simulation-test correlation：把仿真结果和实测数据闭环校准，避免 package / board 级问题只在后期暴露。

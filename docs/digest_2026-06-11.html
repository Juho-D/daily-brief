关键术语小词典

System Technology Co-Optimization

把 silicon、package、board、thermal、power、test 和 workload 一起优化，而不是各团队顺序交接。

Panel-Level Packaging

用面板而不是圆片进行封装加工，目标是降低成本和提高吞吐，但 warpage、overlay 和 RDL uniformity 更难。

SOCAMM2

面向紧凑高带宽 memory module 的形态之一，与 LPDDR 向 AI server 扩展相关；重点看 connector/channel/thermal。

Inline 3D X-ray / CT

用于在封装内部寻找隐藏缺陷的非破坏性检测方法，关键指标是 resolution、throughput 与误检率。

PDN Transient in CIM

存算一体同时激活大量 memory/compute 单元时引发的供电瞬态，可能抵消架构层加速收益。

关键术语小词典

EMIB-T

Intel EMIB 的 TSV bridge 方向，用 embedded bridge 缩短信号/电源路径，重点看 bridge parasitic、PDN 和热机械可靠性。

TGV

Through-glass via。玻璃基板路线的核心垂直互连，优点是高频损耗与尺寸潜力，难点是量产、warpage 和可靠性。

CPO / CPC

CPO 把光引擎靠近 ASIC；CPC 用更近距离铜互连支撑 scale-up/scale-inside。两者需要按 reach、功耗、维护性、延迟和热约束比较。

UCIe compliance

不只是 PHY eye，还包括 protocol、sideband、power state、security、interoperability 和 package channel 的系统测试。

SOCAMM2

面向 AI server memory hierarchy 的 LPDDR 类模块路线，重点看容量、带宽、连接器通道、散热和平台生态。

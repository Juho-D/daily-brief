关键术语小词典

  • W2W / D2W hybrid bonding：wafer-to-wafer / die-to-wafer 混合键合；前者利于高密度和对准一致性，后者更适合异构 die 与 known-good-die 流程。
  • Overlay accuracy：上下层互连对准精度，直接影响 fine-pitch bonding 的连通、短路和良率。
  • τ Scaling Law：华为提出的 scaling 叙事，强调压缩信号传播时间常数，而非只依赖晶体管几何缩小。
  • LogicFolding：把逻辑布局从传统二维平面折叠到三维结构，以缩短关键路径和降低 RC 负载的架构思路。
  • SDA flow：system design and analysis flow，用于把芯片、封装、板级、热、电源和信号分析纳入同一设计闭环。
  • ELS：External Laser Source，CPO 中把激光源外置以改善热、可靠性或可维护性的方案之一。

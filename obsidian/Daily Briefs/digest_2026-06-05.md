## 今日重点判断

1. CPO 已经不是“有没有”的问题，而是“怎么量产验证”：NVIDIA、TSMC COUPE 和 optics 供应链同时出现，评价重点要转向 laser source、SerDes margin、thermal drift、BER、pJ/bit、repair/test 和 switch-level reliability。
2. HBM4/HBM4E 的新闻要按 advanced packaging 读：更高 pin speed 和更大 stack capacity 会直接放大 package PDN、microbump/hybrid bonding、warpage、thermal resistance、die-to-die timing 和 system SI margin 的难度。
3. UCIe 与 3DIC flow 的核心是仿测结合：reference channel、IBIS-AMI/full-wave extraction、PDN noise、thermal map 和 compliance telemetry 没闭环，速率数字会变成纸面优势。

## 关键词小词典

- CPO / Co-Packaged Optics：把 optical engine 放到 switch ASIC 附近，减少高损耗 electrical channel，但引入 laser、thermal、fiber attach、test/repair 难题。
- COUPE：TSMC 面向 optical/electrical co-packaging 的先进封装平台叙事，重点是和 ASIC/package/system 共同验证。
- UCIe：Universal Chiplet Interconnect Express。它不是只有 PHY 速率，还包括 package channel、protocol、compliance、DFx 和 telemetry。
- SOCAMM2：面向 AI server memory hierarchy 的 compact memory module 方向，常与 LPDDR-style 高密度低功耗内存讨论在一起。
